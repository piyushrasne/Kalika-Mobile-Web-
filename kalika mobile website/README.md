# Kalika Mobile Website

A premium, responsive e-commerce website for mobile accessories, featuring a dynamic hero slider, product filtering, and an admin panel.

## Features
- **Hero Slider**: Dynamic, touch-responsive slider with administrative controls.
- **Product Grid**: Filter by category, price, and search (text & image).
- **Admin Panel**: Manage products, categories, and slider content (PIN: 1234).
- **Dark/Light Mode**: User preference persistence.
- **Analytics**: Admin-only visitor tracking.

## Deployment to Netlify

### Option 1: Drag & Drop
1.  Log in to [Netlify](https://app.netlify.com/).
2.  Drag this entire project folder onto the "Sites" dashboard.
3.  Netlify will deploy it instantly.

### Option 2: via GitHub (Recommended)
1.  Push this code to a GitHub repository.
2.  Log in to Netlify and click "Add new site" > "Import from an existing project".
3.  Connect GitHub and select your repository.
4.  **Build Settings**:
    - **Build Command**: (Leave empty)
    - **Publish Directory**: `/` (or leave empty)
5.  Click **Deploy**.

## Admin Access
- URL: `/admin.html`
- Default PIN: `1234`
